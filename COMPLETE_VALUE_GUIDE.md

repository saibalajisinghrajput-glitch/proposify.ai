## üóÑÔ∏è STEP 1: MONGODB ATLAS CONNECTION STRING

### Go to MongoDB Atlas:
**URL**: https://cloud.mongodb.com

### If you need to login:
1. **Click**: "Log In"
2. **Enter**: Your email and password
3. **Click**: "Log In"

### Navigate to Clusters:
1. **Look for**: "Clusters" in the left sidebar menu
2. **Click**: "Clusters"
3. **You'll see**: A list of your database clusters

### Get Connection String:
1. **Find**: Your cluster (usually named "Cluster0")
2. **Check**: Status shows "Available" (green)
3. **Click**: "Connect" button (blue button on the right)
4. **Select**: "Connect your application" option
5. **Copy**: The connection string (looks like):
   ```
   mongodb+srv://<username>:<password>@cluster0.xxxxx.mongodb.net/?retryWrites=true&w=majority
   ```

### Customize the String:
1. **Replace**: `<username>` with your database username
2. **Replace**: `<password>` with your database password
3. **Add**: `proposify` to the end (before `?`)

**Final format**:
```
mongodb+srv://proposifyuser:YourPassword@cluster0.xxxxx.mongodb.net/proposify?retryWrites=true&w=majority
```

**This is your MONGODB_URI! ‚úÖ**
