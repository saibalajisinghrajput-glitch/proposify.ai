## üîê STEP 1: LOGIN

## üìä STEP 2: FIND YOUR CLUSTER
1. **Look for**: "Clusters" in the left sidebar
2. **Click**: "Clusters"
3. **You'll see**: A list of your database clusters
4. **Find**: Your cluster (usually named "Cluster0" or similar)
5. **Look for**: A green "Available" status indicator

## üîå STEP 3: GET CONNECTION STRING
1. **Find**: Your cluster in the list
2. **Click**: The "Connect" button (blue button on the right)
3. **A popup will appear** with connection options
4. **Click**: "Connect your application"
5. **You'll see**: A connection string that looks like this:
   ```
   mongodb+srv://<username>:<password>@cluster0.xxxxx.mongodb.net/?retryWrites=true&w=majority
   ```

---

## ‚úèÔ∏è STEP 4: CUSTOMIZE THE STRING
1. **Copy**: The connection string from the popup
2. **Replace**: `<username>` with your database username (usually "proposifyuser")
3. **Replace**: `<password>` with your database password (the one you created when setting up the database)

**Final result should look like**:
```
mongodb+srv://proposifyuser:YourPassword123@cluster0.abcd.mongodb.net/proposify?retryWrites=true&w=majority
```

## üö® COMMON ISSUES

**Can't find Clusters section**:
- Make sure you're logged in
- Look for "Clusters" in the left navigation menu

**Cluster shows "Creating"**:
- Wait 2-3 minutes for it to become "Available"
- Try refreshing the page

**Connection string has placeholders**:
- Replace `<username>` with your actual database username
- Replace `<password>` with your actual database password
- These were created when you first set up the database

**Don't have a database user**:
1. Go to "Database Access" in left sidebar
2. Click "Add New Database User"
3. Create username: `proposifyuser`
4. Create a secure password
5. Set privileges to "Atlas admin"
6. Click "Add User"
