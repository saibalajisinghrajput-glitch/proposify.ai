## üóÑÔ∏è 1. MONGODB ATLAS CONNECTION STRING

### Step 1: Login to MongoDB Atlas
1. **Go to**: https://cloud.mongodb.com
2. **Login** with your account

### Step 2: Get Connection String
1. **Click**: "Clusters" in left sidebar
2. **Find your cluster** (usually named "Cluster0" or similar)
3. **Click**: "Connect" button (blue button)
4. **Choose**: "Connect your application"
5. **Copy**: The connection string (it looks like this):
   ```
   mongodb+srv://<username>:<password>@cluster0.xxxxx.mongodb.net/?retryWrites=true&w=majority
   ```
6. **Replace**: `<username>` and `<password>` with your actual database credentials

**Final format should be**:
```
mongodb+srv://proposifyuser:yourpassword@cluster0.xxxxx.mongodb.net/proposify?retryWrites=true&w=majority
```
